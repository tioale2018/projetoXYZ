A pasta "vendor/" será gerada automaticamente ao rodar o Composer.
Siga estas instruções para configurar:

1. Certifique-se de ter o Composer instalado em sua máquina.
2. Navegue até o diretório principal do projeto (onde está o "composer.json").
3. Execute o comando abaixo no terminal:
   composer install

Isso criará a pasta "vendor/" e os arquivos necessários para o autoload.
